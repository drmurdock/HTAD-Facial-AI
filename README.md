# HTAD Facial AI

HTAD Facial AI is an AI-powered facial phenotyping platform for hereditary thoracic aortic disease (HTAD) syndromes.

The current version is built upon and adapted from the [PhenoScore](https://github.com/ldingemans/PhenoScore) framework to support gene-specific facial analysis and research into syndromic vascular disorders.
